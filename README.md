# simple-react-native
FoodNote to save image, name, description, price from what you ate.
## Getting Started
To install please run
```
git clone https://github.com/wahfebrianto/simple-react-native.git
cd simple-react-native
npm install
```
## Deployment
To build please run
```
npm install -g expo-cli
expo build:android
expo build:ios
```
